# Heurísticas Compartilhadas - AgroClaudia

## Heurísticas Gerais

### Visibilidade
- O sistema mostra o que está acontecendo?
- Feedbacks, loading, erros são claros?
- O usuário sabe o estado atual do sistema?

### Consistência
- Os mesmos elementos se comportam igual em contextos diferentes?
- A linguagem visual é consistente?
- Padrões de interação são previsíveis?

### Recuperação
- Quando algo dá errado, o usuário consegue voltar a um estado seguro?
- Existe sempre um caminho de volta?
- Erros não deixam o usuário preso?

### Confiança nos Dados
- O que é exibido corresponde ao que foi inserido?
- Dados são persistidos corretamente?
- Não há corrupção ou perda de informação?

### Boundary Testing
- 0, 1, máximo, nulo, especial — o sistema resiste?
- Limites são respeitados?
- Valores extremos são tratados adequadamente?

### Fluxo de Erro
- Erros têm mensagem útil?
- O usuário sabe o que fazer a seguir?
- Mensagens de erro são claras e acionáveis?

### Consistência Visual
- Botões, labels e hierarquia visual fazem sentido?
- Cores e tamanhos comunicam importância?
- Layout é intuitivo?

## Heurísticas Específicas para E-commerce

### Catálogo de Produtos
- Imagens carregam corretamente?
- Informações de produto são completas?
- Preços são exibidos corretamente?
- Categorias são claras e úteis?

### Navegação
- O usuário sempre sabe onde está?
- Breadcumbs funcionam?
- Busca retorna resultados relevantes?
- Filtros são intuitivos?

### Checkout/Contato
- Fluxo é claro e simples?
- Passos são bem definidos?
- Feedback é dado em cada etapa?
- Erros são fáceis de corrigir?

### Performance
- Carregamento é rápido o suficiente?
- Transições são suaves?
- Não há travamentos ou lentidão?
- Imagens são otimizadas?

### Responsividade
- Funciona em mobile?
- Funciona em tablet?
- Funciona em desktop?
- Layout se adapta corretamente?

## Heurísticas Específicas para AgroClaudia

### WhatsApp Integration
- Botão de WhatsApp é visível?
- Link abre corretamente?
- Mensagem pré-preenchida faz sentido?
- Funciona em mobile?

### Categorias Agro
- Categorias agropecuária estão claras?
- Imagens representam corretamente as categorias?
- Navegação entre categorias é fluida?
- Busca por categoria funciona?

### Contato e Informações
- Informações de contato estão corretas?
- Links sociais funcionam?
- Formulário de contato é funcional?
- Endereço/Localização está correto?
